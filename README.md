## ðŸ›  Code Reference

### Core TIPMOC Functions
| File | Description |
| :--- | :--- |
| `fit_powerlaw.py` | Fits the specific power-law diverging function to a sequence of ($u$, $\hat{V}$). |
| `delta_aicc.py` | Computes $\Delta \text{AIC}_{\text{c}}$ (difference between linear and power-law fits). |

### Simulation Runners
| File | System / Model |
| :--- | :--- |
| `run_double_well.py` | Double-well dynamical system. |
| `run_over_harvesting.py` | Over-harvesting models ([Kefi et al., 2013](https://doi.org/10.1111/j.1600-0706.2012.20838.x)). |
| `run_mutualistic.py` | Mutualistic-interaction model on a network. |
| `run_ou.py` | Ornstein-Uhlenbeck process. |
| `test_tipmoc_100_1.py` | Batch runner to produce data for Figures 2(a), 2(b), and Table 1. |

### Visualization & Analysis
| File | Figure Produced |
| :--- | :--- |
| `explain_tipmoc.py` | Schematic (Figure 1). |
| `one_run_plot_using_given_data.py` | Fine-tuned Figures 2(a) and 2(b), using numerical data files generated by `test_tipmoc_100_1.py`. |
| `sample-plot-using-given-data.py` | Fine-tune Figures 2(c) and 4, using numerical data files generated by `sample-plot.py`.|
| `sample-plot.py` | Produce data `sample-plot-using-given-data.py` uses and plot preliminary versions Figures 2(c) and 4.|
| `plot_hatuc_scatter_dw.py` | Diagnostic scattergrams (Figures 3(a) and 3(b)). |
| `diagnostic_analysis.py` | Summary statistics for Table 1, computed from the simulated data produced by `test_topmoc_100_1.py`.|

### Auxiliary Scripts
* `bifu_pt_double_well.py`: Determines bifurcation point (3.079) for the double-well model.
* `bifu_pt_mutualistic.py`: Determines collapse point (0.0470) for the mutualistic-interaction model.
